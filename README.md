\# Attendance-APP-apk



Attendance Management Android Application built using Java and Firebase.



\## Description

This project is an Android app that allows teachers to manage subjects and mark student attendance.  

It includes separate dashboards for teachers and students and stores data using Firebase Firestore.



\## Features

\- User Login \& Registration

\- Teacher Dashboard

\- Student Dashboard

\- Add and manage subjects

\- Mark attendance using RecyclerView

\- Attendance by date

\- Firebase Firestore database integration



\## Technologies Used

\- Android Studio

\- Java

\- Firebase Firestore

\- XML (UI Design)

\- RecyclerView \& Adapters



\## How to Run

1\. Clone the repository:

git clone https://github.com/talhazubairr/Attendance-APP-apk.git

2\. Open the project in Android Studio

3\. Let Gradle sync completely

4\. Connect Firebase and add `google-services.json` inside the `app` folder

5\. Run the project on emulator or physical device
